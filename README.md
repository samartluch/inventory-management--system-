# Inventory Mamagement System 

## Description
A Flask web application for managing products, stock, and suppliers.

## Technologies
- Python (Flask)
- SQLite (Database)
- Tailwind CSS (Frontend)

## How to Run
1. Clone the repository  
   `git clone https://github.com/username/inventory-system.git`
2. Create virtual environment  
   `python -m venv venv`
3. Install dependencies  
   `pip install -r requirements.txt`
4. Run the app  
   `python run.py`
